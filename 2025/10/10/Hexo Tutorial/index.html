Hexo Tutorial

Word Count:   725
Read Times:   3 Min
Read Count:  

此md文档为使用hexo框架搭建个人博客网站的详细教程。
常用命令主要有四个hexo clean，hexo g，hexo s，hexo d，其他的用的比较少
修改的主要配置文件有三个，一个是BLOG/目录下的_config.yml和_config.butterfly.yml配置文件,一个是BLOG/themes/hexo-theme-matery/目录下的_config.yml配置文件
建议把所有的图片、视频、文档、样式等全部放入BLOG/source/下进行管理，可以在BLOG/source/下建很多个文件夹分类存放资源

参考教程

https://blog.csdn.net/qq_22841387/article/details/151066378

【hugo suggestions】

1.hexo框架搭建：https://hexo.io/zh-cn/
2.建站：建立在github上 http://hjzts.github.io 买个域名，在阿里云（腾讯云）网站上做个转发
3.选择themes主题：matery、next(魔改版比较多) ，去找一些hexo美化教程
配置：樱花、点击特效、个性化文字、图片、看板娘、音乐
html转md：gpt
图片可以用图床：1h，可有可无
资料分享，用个云盘
评论功能：30min

博客园：几乎全部配置好了，30min可以全部搞定

https://volcaxiao.top/
https://cutedian.top/
https://yanna-zy.github.io/

【hexo file level】

your-hexo-site/ ← 项目根（任意名字）
├── _config.yml 【站点配置】
├── _config.landscape.yml 【主题独立配置文件（hexo 5+ 推荐方式）】
├── db.json 【索引缓存】
├── debug.log 【报错时自动生成】
├── package.json 【npm 依赖】
├── package-lock.json
├── node_modules/ 【npm 装进来的包】
│ ├── hexo/
│ ├── hexo-cli/
│ ├── hexo-generator-feed/
│ └── …(100+ 文件夹，略)
├── scaffolds/ 【文章/页面模板】
│ ├── post.md
│ ├── draft.md
│ ├── page.md
│ └── photo.md ← 自己加的自定义模板
├── source/ 【用户源文件】
│ ├── _posts/ 【文章】
│ │ ├── hello-world.md ← hexo 默认第一篇
│ │ ├── 2025-10-10-深度研究.md
│ │ └── 2025/ ← 你也可以再建子文件夹归类
│ │ └── 10/
│ │ └── 我的十月总结.md
│ ├── _drafts/ 【草稿】
│ │ ├── todo.md
│ │ └── 未完待续.md
│ ├── images/ 【全局静态资源】
│ │ ├── logo.png
│ │ └── 2025/
│ │ └── banner.jpg
│ ├── about/ 【自定义页面】
│ │ └── index.md
│ ├── downloads/ ← 你自己建的任意文件夹
│ │ └── pdf/
│ │ └── resume.pdf
│ └── CNAME ← 放这会被原样拷到 public/（GitHub Pages 自定义域名）
├── themes/ 【主题】
│ ├── landscape/ 【默认主题】
│ │ ├── _config.yml 【主题配置】
│ │ ├── layout/ 【ejs/pug 模板】
│ │ │ ├── _partial/
│ │ │ ├── _widget/
│ │ │ ├── archive.ejs
│ │ │ ├── index.ejs
│ │ │ ├── post.ejs
│ │ │ └── page.ejs
│ │ ├── source/ 【主题静态资源】
│ │ │ ├── css/
│ │ │ ├── js/
│ │ │ └── images/
│ │ └── scripts/ 【主题注入脚本】
│ └── next/ ← 你后来 git clone 的流行主题
│ ├── _config.yml
│ ├── layout/
│ ├── source/
│ └── languages/
├── scripts/ 【Hexo 插件级脚本】
│ ├── word-count.js ← 自己写的字数统计
│ └── image-resize.js
└── public/ 【hexo g 后生成，真正部署出去的网站】
├── index.html
├── archives/
├── 2025/
│ └── 10/
│ └── 我的十月总结/
│ └── index.html
├── about/
│ └── index.html
├── images/
│ └── logo.png
├── css/
├── js/
└── atom.xml
