Hexo Tutorial

hexo command hexo file level hugo
Publish Date:   2025-10-10
Update Date:   2026-01-20
Word Count:   1.1k
Read Times:   4 Min

此md文档为使用hexo框架搭建个人博客网站的详细教程。
常用命令主要有三个hexo clean，hexo g，hexo s，其他的用的比较少
修改的主要配置文件有三个，一个是BLOG/目录下的_config.yml和_config.butterfly.yml配置文件,一个是BLOG/themes/hexo-theme-matery/目录下的_config.yml配置文件
建议把所有的图片、视频、文档、样式等全部放入BLOG/source/下进行管理，可以在BLOG/source/下建很多个文件夹分类存放资源

参考教程

https://blog.csdn.net/qq_22841387/article/details/151066378

【hexo command】

1. 安装与升级

npm install hexo-cli -g
npm update -g hexo-cli

2. 初始化 & 依赖

hexo init
npm install
npm audit fix

3. 本地预览

hexo server
hexo s -p 5000
hexo s -o

4. 生成 & 清理

hexo generate
hexo g
hexo g –watch
hexo clean
hexo clean && hexo g

5. 部署（一键发布）

hexo deploy
hexo d
hexo g -d

6. 文章 / 页面 / 草稿

hexo new post “My Post”
hexo new page “About”
hexo new draft “Secret”
hexo publish draft “Secret”

7. 列表 & 信息

hexo list post
hexo list page
hexo list route
hexo list tag
hexo list category

8. 性能与调试

hexo –draft
hexo –safe
hexo –debug

9. 主题 & 插件

git clonethemes/
npm install–save
npm update

10. 一键组合别名（可写进 package.json）

“scripts”: {
“dev”: “hexo clean && hexo g && hexo s”,
“build”: “hexo clean && hexo g”,
“deploy”: “hexo clean && hexo g && hexo d”
}

11. 使用：

npm run dev
npm run build
npm run deploy

【hexo file level】

your-hexo-site/ ← 项目根（任意名字）
├── _config.yml 【站点配置】
├── _config.landscape.yml 【主题独立配置文件（hexo 5+ 推荐方式）】
├── db.json 【索引缓存】
├── debug.log 【报错时自动生成】
├── package.json 【npm 依赖】
├── package-lock.json
├── node_modules/ 【npm 装进来的包】
│ ├── hexo/
│ ├── hexo-cli/
│ ├── hexo-generator-feed/
│ └── …(100+ 文件夹，略)
├── scaffolds/ 【文章/页面模板】
│ ├── post.md
│ ├── draft.md
│ ├── page.md
│ └── photo.md ← 自己加的自定义模板
├── source/ 【用户源文件】
│ ├── _posts/ 【文章】
│ │ ├── hello-world.md ← hexo 默认第一篇
│ │ ├── 2025-10-10-深度研究.md
│ │ └── 2025/ ← 你也可以再建子文件夹归类
│ │ └── 10/
│ │ └── 我的十月总结.md
│ ├── _drafts/ 【草稿】
│ │ ├── todo.md
│ │ └── 未完待续.md
│ ├── images/ 【全局静态资源】
│ │ ├── logo.png
│ │ └── 2025/
│ │ └── banner.jpg
│ ├── about/ 【自定义页面】
│ │ └── index.md
│ ├── downloads/ ← 你自己建的任意文件夹
│ │ └── pdf/
│ │ └── resume.pdf
│ └── CNAME ← 放这会被原样拷到 public/（GitHub Pages 自定义域名）
├── themes/ 【主题】
│ ├── landscape/ 【默认主题】
│ │ ├── _config.yml 【主题配置】
│ │ ├── layout/ 【ejs/pug 模板】
│ │ │ ├── _partial/
│ │ │ ├── _widget/
│ │ │ ├── archive.ejs
│ │ │ ├── index.ejs
│ │ │ ├── post.ejs
│ │ │ └── page.ejs
│ │ ├── source/ 【主题静态资源】
│ │ │ ├── css/
│ │ │ ├── js/
│ │ │ └── images/
│ │ └── scripts/ 【主题注入脚本】
│ └── next/ ← 你后来 git clone 的流行主题
│ ├── _config.yml
│ ├── layout/
│ ├── source/
│ └── languages/
├── scripts/ 【Hexo 插件级脚本】
│ ├── word-count.js ← 自己写的字数统计
│ └── image-resize.js
└── public/ 【hexo g 后生成，真正部署出去的网站】
├── index.html
├── archives/
├── 2025/
│ └── 10/
│ └── 我的十月总结/
│ └── index.html
├── about/
